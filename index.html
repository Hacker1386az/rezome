درباره من

دانش‌آموز پایه یازدهم رشته ریاضی فیزیک، با تجربه عملی در طراحی انواع وب‌سایت‌های تجاری و شخصی. مسلط به ابزارهای طراحی مدرن، علاقه‌مند به طراحی خلاقانه، توسعه وب‌سایت‌های واکنش‌گرا و بهینه‌سازی برای موتورهای جستجو. توانایی کار مستقل و تیمی و علاقه‌مند به یادگیری مداوم.

مهارت‌ها

  • طراحی صفحات وب با HTML5 و CSS3
  • طراحی واکنش‌گرا (Responsive) با Bootstrap و Tailwind CSS
  • JavaScript و jQuery
  • پیاده‌سازی و طراحی قالب در WordPress
  • Git و GitHub
  • طراحی رابط کاربری (UI) و تجربه کاربری (UX)
  • آشنایی با SEO و بهینه‌سازی
  • آشنایی با React.js
  • کار با لینوکس (Kali Linux) و مفاهیم امنیت سایبری
  • ساخت ربات تلگرامی و ابزارهای تست نفوذ با Python

پروژه‌ها

  1. سایت کافه: طراحی سایت معرفی و منوی کافه با UI دل‌نشین
  2. سایت خبری: ساخت سایت خبری با html , css , js
  3. سایت حسابداری: طراحی سایت معرفی خدمات حسابداری
  4. سایت فروشگاهی: طراحی فروشگاه پوشاک با درگاه پرداخت
  5. سایت آموزشی: پیاده‌سازی سیستم آموزش آنلاین
  6. سایت موزیک: سایت انتشار آهنگ با پخش آنلاین
  7. سایت بازسازی ساختمان و پیمانکاری: سایت معرفی پروژه‌ها، نمونه‌کارها و فرم درخواست خدمات
